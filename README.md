# MovieChecker
by John Hinrichs

### Overview

**MovieChecker** is an app I'm building to learn Backbone.js which uses the Rotten Tomatoes API.

### Technologies Used

* Backbone.js 1.0.0
* Underscore.js 1.6.0
* Ruby on Rails 4.0.4
* PostgreSQL Database
* Rotten Tomatoes API

[View this project on Heroku](http://moviechecker.herokuapp.com/)
